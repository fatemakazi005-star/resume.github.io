# 🌟 Fatema Mahamad Kazi — Digital Portfolio

**Live Site:** `https://fatema-kazi.github.io/portfolio`

A professional student digital portfolio showcasing achievements, projects, certificates, and more.  
Built with pure **HTML · CSS · JavaScript** — no frameworks or external builders.

---

## 📁 Folder Structure

```
portfolio/
├── index.html          ← Main HTML file
├── style.css           ← All styles (dark/light theme, animations)
├── script.js           ← Interactions, typed text, scroll reveal, etc.
├── README.md           ← This file
└── assets/             ← Put ALL your images here
    ├── profile.jpeg        ← Your profile photo
    ├── cert-yuva-ai.jpg    ← Yuva AI certificate image
    ├── cert-modern-ai.jpg  ← Modern AI certificate image
```

- 📸 **Scroll Reveal Animations** — smooth fade-in on scroll

## 🚀 How to Deploy on GitHub Pages

1. Create a new repository named `portfolio` on GitHub
2. Upload all these files (keeping folder structure intact)
3. Go to **Settings → Pages**
4. Set Source to **main branch / root**
5. Your site will be live at `https://YOUR-USERNAME.github.io/portfolio`

---

## 🖼️ Adding Your Images

1. Create an `assets/` folder in your repository
2. Upload your actual photos and certificate scans there
3. Make sure filenames match exactly (see folder structure above)
4. If a photo is missing, the site gracefully shows a placeholder icon
